# luaSTG+

luaSTGPlus项目致力于发展luastg的后续可维护版本。

本项目以**LuaSTG.2012.8源代码(by 隔壁的桌子)**为基准，在保证**大多数功能兼容**的前提下重写luaSTG底层框架，以优化完善过去老的代码、适应新的游戏开发要求为目标。

## API文档

请查阅`APIManual`以获得更多帮助。

## luaSTG代码迁移

请查阅`luastg迁移向导`以获得更多帮助。
